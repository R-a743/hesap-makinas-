# 📱 Basit Hesap Makinesi

Bu proje, HTML, CSS ve JavaScript kullanılarak oluşturulmuş basit bir hesap makinesi uygulamasıdır. Toplama, çıkarma, çarpma ve bölme gibi temel matematiksel işlemleri gerçekleştirebilir.

---

## 🖼️ Demo

Projeyi canlı olarak görmek için: [Buraya Git](https://r-a743.github.io/hesap-makinas-/)  


---

## 🛠️ Kullanılan Teknolojiler

- **HTML** – Uygulamanın iskelet yapısı
- **CSS** – Stil ve tasarım
- **JavaScript** – Hesap makinesi işlevleri ve kullanıcı etkileşimleri

---

## ⚙️ Özellikler

- Sayılarla işlem yapma
- Toplama, çıkarma, çarpma, bölme
- Temizle (C) butonu ile ekranı sıfırlama
- Modern ve sade arayüz
